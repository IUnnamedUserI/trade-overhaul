# trade-overhaul

## Описание

Trade Overhaul - это мод для Minecraft, который полностью перерабатывает систему торговли с жителями. Мод предоставляет расширенный интерфейс торговли и гибкую систему настройки торговых предложений.

## Особенности

- **Расширенный интерфейс торговли**: Более удобный и информативный интерфейс для взаимодействия с жителями
- **Настраиваемые торговые предложения**: Возможность настройки предложений для разных профессий жителей
- **Пакетная торговля**: Позволяет покупать и продавать товары партиями по фиксированному соотношению (например, 1 изумруд за 10 морковок)
- **Гибкая система ценообразования**: Поддержка различных типов товаров (статические, по тегам, инструменты, оружие)
- **Конфигурируемые параметры**: Все аспекты торговли могут быть настроены через JSON-файлы

## Пакетная торговля

Одной из ключевых особенностей мода является система пакетной торговли, которая решает проблему неэффективной экономики в ванильной игре:

- Игрок может купить 10 морковок за 1 изумруд одним кликом
- Игрок может продать 8 морковок за 1 изумруд одним кликом
- Количество товаров в инвентаре жителя всегда кратно размеру пакета
- Поддержка массовых операций через Shift+клик

## Конфигурация

Файлы конфигурации находятся в папке `config/tradeoverhaul/professions/`. Каждый файл соответствует определенной профессии жителя и содержит настройки торговых предложений.

Пример конфигурации:
```json
{
  "profession": "minecraft:farmer",
  "offersCount": 7,
  "staticPool": [
    { 
      "item": "minecraft:carrot", 
      "buy": 1, 
      "sell": 1, 
      "buyQuantity": 15,    // Игрок получает 15 морковок за 1 изумруд
      "sellQuantity": 8,    // Игрок продает 8 морковок за 1 изумруд
      "minStock": 15, 
      "maxStock": 45 
    }
  ]
}
```

## Установка

1. Установите Fabric Loader
2. Установите Fabric API
3. Поместите JAR-файл мода в папку `mods`
4. Запустите игру для генерации файлов конфигурации

## Совместимость

Мод совместим с другими модами, использующими систему Mixin. Имеет интеграцию с системой досок объявлений (Bulletin Board) для дополнительных возможностей.
